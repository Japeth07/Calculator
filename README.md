# Light/Dark Mode Calculator

A sleek, web-based calculator that combines functional arithmetic with a modern, themeable interface. Built using pure vanilla web technologies, it allows users to perform calculations while switching between light and dark modes via intuitive icons.

---

## Features

### Arithmetic Operations
- **Basic Math:** Performs Addition (`+`), Subtraction (`-`), Multiplication (`*`), and Division (`/`).
- **Precision:** Handles floating-point calculations with ease.

### Calculator Controls
- **C (Clear):** Instantly clears the entire input and current calculation.
- **DEL (Delete):** Removes the last character entered for quick corrections.
- **= (Equals):** Evaluates the expression and displays the final result.

### Theming
- **Mode Toggle:** Seamlessly switch between themes using visual icons:
  - **Light Mode:** High-contrast, bright interface for daytime use.
  -  **Dark Mode:** Easy-on-the-eyes, low-light interface for nighttime use.
- **Persistent Logic:** Maintains the current calculation even when swapping themes.
